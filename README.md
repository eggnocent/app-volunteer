# Migunani

Migunani adalah prototype frontend marketplace event volunteer untuk mahasiswa, relawan umum, dan organisasi penyelenggara event.

Stack:

- React + Vite
- TypeScript
- Tailwind CSS
- React Router
- Lucide React
- Framer Motion
- Static dummy data

Run locally:

```bash
nvm use 22
npm install
npm run dev
```

Build check:

```bash
npm run build
npm run lint
```
